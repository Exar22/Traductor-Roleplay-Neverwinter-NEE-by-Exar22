# Traductor-Roleplay-Neverwinter-NEE-by-Exar22
Esta es una aplicacion para poder traducir el chat del juego de neverwinter de ingles al español automaticamente para poder hacer roleplay en cualquier de sus servidores.
⚠️ Nota de compatibilidad: Si juegas a Neverwinter Nights ejecutando el juego como Administrador, recuerda ejecutar también el traductor como Administrador (Clic derecho -> Ejecutar como Administrador) para que Windows no bloquee las funciones de teclado automático de la aplicación (PyAutoGUI).
